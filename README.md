# 日记任务计时

一个面向“今天”的思源笔记侧栏插件：创建今日日记任务、记录番茄专注、完成时选择实际用时，并用简单图表汇总今日总时长。

## 功能

- 右侧栏分为“任务列表 / 今日总结”，顶部日历图标可打开全屏日历。
- 支持一次创建多个任务：新建弹窗中一行一个任务。
- 支持习惯任务：点击刷新时读取指定习惯文档，每行补齐为今日任务。
- 创建任务后追加到所选笔记本下已有的当天日记，不会自动创建文档。
- 任务列表与当天日记双向同步：文档保存后更新一次列表，列表操作会回写对应任务行。
- 任务列表展示今天所有任务，支持完成、开始番茄、放弃、恢复等常用操作。
- 番茄钟为正向计时：选择任务开始专注，活动中的计时只保存在当前端，点击“停止并记录”后才写入任务文档。
- 完成任务时支持两种用时来源：
  - 直接填写总分钟数。
  - 使用该任务累计番茄时间。
- 完成任务时可选填任务总结。
- 日历可按周 / 月 / 年查看所有任务。
- 支持对同一个任务多次开始番茄专注。
- 显示今日已完成任务总时长、番茄累计、饼图分布和每个任务用时；进行中的番茄也会计入当前累计。

## 使用

1. 启用插件后，在右侧 Dock 可切换“任务列表 / 今日总结”，点击顶部日历图标打开全屏日历。
2. 在插件设置里选择日记笔记本；插件会自动寻找该笔记本下已有的当天日记。如需习惯任务，填写习惯文档 ID。
3. 新建任务时一行写一个，可一次创建多个。
4. 保存任务文档后列表会同步一次；点击刷新时会同步文档并补齐习惯文档里的任务。
5. 任务完成后选择用时来源，可选填总结，插件会记录完成状态并向当天日记追加完成记录。

## 数据说明

插件数据保存在插件自己的 `time-list-data.json` 中；当天日记中追加的是可读记录，已完成的番茄明细会写入任务块属性用于跨端同步。
