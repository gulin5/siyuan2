# 日记任务计时

一个面向“今天”的思源笔记侧栏插件：创建今日日记任务、记录番茄专注、完成时选择实际用时，并用简单图表汇总今日总时长。

## 功能

- 右侧栏分为“任务列表”和“今日总结”。
- 支持一次创建多个任务：新建弹窗中一行一个任务。
- 支持习惯任务：每天读取指定习惯文档，每行自动生成今日任务。
- 创建任务后自动追加到指定笔记本的今日日记。
- 任务列表展示今天所有任务，支持完成、开始番茄、放弃、恢复等常用操作。
- 番茄钟为正向计时：选择任务开始专注，停止时记录实际用时。
- 完成任务时支持两种用时来源：
  - 直接填写总分钟数。
  - 使用该任务累计番茄时间。
- 支持对同一个任务多次开始番茄专注。
- 显示今日已完成任务总时长、番茄累计、饼图分布和每个任务用时。

## 使用

1. 启用插件后，在右侧 Dock 可切换“任务列表 / 今日总结”。
2. 在插件设置里选择要写入日记的笔记本；如需习惯任务，填写习惯文档 ID。
3. 新建任务时一行写一个，可一次创建多个。
4. 刷新或重新打开插件时，会为今天补齐习惯文档里的任务。
5. 任务完成后选择用时来源，插件会记录完成状态并向今日日记追加完成记录。

## 数据说明

插件数据保存在插件自己的 `time-list-data.json` 中；日记文档中追加的是可读记录，方便回顾。
